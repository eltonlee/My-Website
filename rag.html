Random Anime Generator

Web application

Introduction

“Random Anime Generator” is a web application that generates a random japanese TV show or movie. It is designed for people who wants to watch anime, but does not know what to watch. This web application will pick a show or movie for them.

Watching anime is one of my favorite hobbies, so I thought it would be fun to create a web application like this.
